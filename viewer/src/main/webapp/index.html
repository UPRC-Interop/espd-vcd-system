ESPD/VCD RI System Documentation

This is the ESPD/VCD Reference Implementation System (v1.0.0-SNAPSHOT) by e-SENS.

logging

logging deals with the logging framework used throughout the system.

common-util

common-util encapsulates all the common functionality that is used throughout the whole system.

compiler

compiler is responsible for the compilation of a document model (i.e. ESPD Request Tmpl).

model

model is responsible for the modeling of all kinds of documents.

persistence

persistence is a JPA implementation in hibernate that is responsible for the persistence of the data in the postgresql database.

schema

schema is the JAXB implementation that generated all Java classes from the corresponding XSD files.

designer

designer is the user interface of the system for both economic operators and contracting authorities.

viewer

viewer is the user interface of the ESPD/VCD Viewer used only by the contracting authorities.
